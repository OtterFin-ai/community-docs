# Tags

Tags are freeform labels you can attach to transactions for tracking purposes that cut across category lines.

## How tags differ from categories

Categories answer "what kind of spending is this?" — groceries, rent, utilities. Tags answer a different question, like "which project is this for?" or "is this a tax-deductible expense?" or "is this part of our vacation spending?"

A transaction has one category (or one category per split line), but it can have multiple tags.

## Common uses

- **Tax tracking** — tag deductible expenses throughout the year so you have an easy summary at tax time
- **Project tracking** — tag all transactions related to a home renovation or a specific trip
- **Reimbursements** — tag expenses you're waiting to be paid back for
- **Shared costs** — tag transactions to track who owes what in a shared household

## Managing tags

Tags are managed in **Settings → Tags** (accessible from the household menu in the sidebar). You can create and rename tags there, or create them on the fly when adding a transaction.

## Filtering by tag

The transaction list can be filtered by tag, letting you pull up all transactions with a given label regardless of account or category.
