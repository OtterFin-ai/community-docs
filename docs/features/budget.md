# Budget

The budget in OtterFin lets you set monthly spending targets by category and track how your actual spending compares against them.

## How budgets work

Budgets are organized by category. You set a target amount for each category you want to monitor, and OtterFin shows how much of that budget you've used based on transactions posted in the current period.

The budget view shows:

- Your target for each budgeted category
- How much you've spent so far in that category this period
- The remaining amount, and whether you're over or under budget

Categories without a budget target are visible in the budget view but are not constrained — transactions still flow through them, they just don't have a set limit.

## Budget periods

Budgets operate on a monthly basis. The budget view defaults to the current month, and you can navigate between months to review past budget performance.

## Setting budget targets

To set a budget target for a category, open the budget view and enter the amount directly inline next to that category. Changes take effect immediately.

## Budget vs. actual

The budget comparison pulls from the same transactions used across the rest of OtterFin. Transactions marked as transfers are excluded. Split transactions are counted toward each category according to the split allocation.
