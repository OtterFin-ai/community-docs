# Settings

OtterFin has two levels of settings: personal settings that apply to your account, and household settings that apply to everyone in the household.

## Profile settings

**Settings → Profile** lets you update your display name and email address.

## Household settings

Household settings are accessible to household owners only.

### General household settings

**Settings → Household** covers:

- **Household name** — the display name for your household
- **Base currency** — the currency used for dashboard totals, net worth, and reports. All multi-currency amounts are converted to this currency for display.
- **Reconciliation policy** — when enabled (the default), transactions can only be marked reconciled through a recorded statement. Disable this if you want to manually mark transactions reconciled without going through the statement workflow.

### Currencies

**Settings → Currencies** manages the currencies your household uses beyond the base currency.

For each currency pair, you can set:

- **Manual exchange rate overrides** — specific rates to use instead of system rates
- **Staleness window** — how many days a manual override stays active before OtterFin falls back to system rates. Set to _never expire_ to always use your override.

See [Household](household.md) for more on how exchange rate resolution works.

### Payment methods

**Settings → Payment Methods** is where you manage the payment method options available when entering transactions — specific credit cards, debit cards, cash, and so on. Payment methods are optional on transactions and are used for filtering and reference only.

### Data

**Settings → Data** provides household-level data management options, including data export.
